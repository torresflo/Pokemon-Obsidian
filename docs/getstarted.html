Download instructions

Head over to the Download page and download the latest version of Pokémon Obsidian.

Also grab 7-Zip if you need software to open compressed archives (like .zip, .rar, .7z, etc.)

Extract the archive to a folder of your choosing. You should obtain several folders and an executable named Obsidian.exe

Setup game

Launch Pokémon Obsidian by launching Obdisian.exe.

Sometimes Pokémon Obsidian triggers a false positive with AntiVirus software. The game is completely safe. It is recommended to make a folder exception to keep your AntiVirus software from deleting or blocking critical files from your Obsidian folder.

As RPG Maker XP is an old software, it is also possible that you need to run the game in a compatible mode if it is not launching correctly. Try right clicking on the executable and selecting the compatibility mode "Windows XP (service pack 2)".

Note for Laptop users:

If you have a Laptop with multiple GPUs (integrated + NVIDIA or AMD), you maybe need to set up the game to force it to use the NVIDIA\AMD GPU. You can do this multiple ways, either in your GPU's Driver, or since Windows 10 1803 (2018 april update) you can force it via Windows Advanced Graphics Settings too.

You can check how to force your GPU in the links below (depending on your GPU, or if you want to force it via driver or windows)
